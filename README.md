# Problems
## String01

  A str of several words is given. All letters are lowercase. Make sure that the first letter of each word is capitalized.

**Example 1:**

```Python
Input: a="google is a search engine"
Output: "Google Is A Search Engine"

```

**Constraints:**

  - 2 <= length(a) <= 10^5
